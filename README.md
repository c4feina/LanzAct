## LanzAct 
Herramienta para abrir el editor, la terminal y el navegador de un proyecto con atajos del teclado. 
(Se puede configurar para lo que quieran)

# Requisitos
Python 3.7+
Sin dependencias externas

# Configuracion
La primera vez que se ejecuta, se crea el archivo de configuracion
Cada proyecto es un objeto JSON con claves opcionales
editor: cualquier comando de shell, se ejecuta tal cual
terminal: ruta de directorio, abre el emulador de terminal ahi
browser: una URL o una lista de URLs

Todas las claves son opcionales. 
Un proyecto puede tener solo terminal, solo navegador, etc.
